# Alpheus tsdown configuration

A shareable tsdown configuration.

## Installation

Install this package as a dependency in the project:

```sh
# npm
npm i @apst/tsdown

# Yarn
yarn add @apst/tsdown

# pnpm
pnpm add @apst/tsdown

# Deno
deno add npm:@apst/tsdown

# Bun
bun add @apst/tsdown
```
